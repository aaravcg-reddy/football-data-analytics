# Football Performance Data Analytics Model

A Python-based data analytics pipeline designed to ingest raw player metrics, filter performance thresholds, and visualize total attacking efficiency.

## Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, Matplotlib

## Key Features
* **Feature Engineering:** Calculated overall attacking contributions by compounding goals and assists.
* **Data Filtering:** Implemented a threshold matrix to isolate elite performers (Match Rating >= 8.0).
* **Data Visualization:** Engineered a stacked bar chart displaying individual metrics.
